# Chrome Web Store listing draft

## Name

Min-Conflit Solver

## Short description

Solve and visualize N-Queens and Sudoku locally in Chrome with an offline Min-Conflict workspace.

## Single purpose

This extension provides one educational workspace for visualizing and comparing local CSP solving strategies, focused on N-Queens and Sudoku.

## Full description

Min-Conflit Solver is a local Chrome extension built for algorithm exploration.

It lets users:

- solve N-Queens with a Min-Conflict strategy
- solve Sudoku with Backtracking or Min-Conflict
- replay each step of the solver
- compare local benchmark results
- run everything locally, without a remote backend

Why it is useful:

- good for students working on CSP, heuristics, and local search
- useful for demos, coursework, and portfolio presentations
- works offline once installed

## Permissions justification
