Privacy Policy for Min-Conflit Solver

Last updated: 2026-05-20

Summary

Min-Conflit Solver runs locally in the browser. Its core solving functionality does not require a remote backend and does not collect browsing history.

What the extension stores locally

  • The selected mode such as N-Queens or Sudoku.
  • The last solver settings used by the visitor.
  • The last custom Sudoku grid entered by the visitor.
  • A short summary of the last run so the interface can restore context.

This information is stored only to preserve the local experience between sessions.

What the extension does not collect

  • No browsing history.
  • No website content from arbitrary tabs.
  • No personal identifiers for analytics or advertising.
  • No remote transmission of user-generated puzzle content for core functionality.

Permissions

The extension currently requests only the storage permission. This is used to save local preferences and last-run context inside the browser.

Remote services

The core product is designed to function without a remote backend. If future versions add optional external services, this page should be updated before publication of that version.
